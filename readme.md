## 功能介绍

一款文字游戏游玩框架，自定义剧情json后可玩



## BUG记录

	1. 剧情块滚动条背景色（白色）未能成功修改，原因不明。（于2025.04.07放弃维护，具体查看对应日期的更新记录）
	2. 剧情块右边框被人物属性面板遮挡。（于2025.04.07解决，具体查看对应日期的更新记录）

## 更新记录

#### 2025.03.27

	1. 添加剧情块的滚动条

#### 2025.04.04

	1. 优化了剧情块的滚动条样式，添加“悬停高亮”功能。
