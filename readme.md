## 功能介绍

一款基于Vue的文字游戏框架，通过自定义JSON剧本即可快速构建剧情分支系统。支持以下特性：
- 可视化侧边栏拖拽调整
- 自动剧情滚动定位
- 角色属性实时更新（通过修改<Character_attributes/attributes.js>文件配置角色初始属性）
- 多分支剧情支持
- 本项目搭载nodejs依赖，请自主安装，否则无法正常运行项目！！
